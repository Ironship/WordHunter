Dane lokalne

Import i eksport

Pełna kopia WordHuntera

JSON zawiera ustawienia, profile językowe, bazę słów, własne teksty i ukryte książki. Import JSON zastępuje aktualny stan aplikacji danymi z pliku.

Wymiana słówek z Anki

TSV eksportuje tylko słowa, tłumaczenia i pierwszy kontekst. Import TSV dopisuje lub aktualizuje słówka, ale nie przenosi ustawień, książek ani profili.

Eksportowane statusy
Wybierz, które statusy słów trafią do TSV. Anki dostaje tylko kolumny: słowo, tłumaczenie i kontekst.

Przywrócenie ustawień

Resetuje motyw, czytnik, słownik, TTS i inne preferencje do wartości domyślnych. Nie usuwa słówek ani tekstów.

Wyczyść bazę słów

Usuwa wszystkie zapisane słowa, tłumaczenia i postęp nauki dla tego języka.

Wyczyść bibliotekę

Usuwa wszystkie własne teksty i resetuje listę książek, zachowując bazę Twoich słów.
