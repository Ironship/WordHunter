# Original starter books

This directory contains two kinds of built-in Word Hunter Originals:

- **Common-word story collections** use short, clear stories about everyday
  situations and target at least 1,000 distinct common lexical items.
- **A1-B2 course books** provide four progressively leveled books per language.
  Each level is one continuous plain-text book with lessons, connected material,
  and practice activities.

The modern-language courses can support CEFR-style exam preparation, but they do
not guarantee an exam result or claim equivalence with any provider-specific
level or test. The Latin and Ancient Greek levels are CEFR-inspired rather than
official CEFR or exam classifications.

All story and course content is original, contains no adapted passages from
existing works, and ships with original SVG cover artwork created for Word
Hunter. These built-in application resources are listed in `../index.json`.
