# Word Hunter 1.0.5-rc.3

> **Warning:** This is a prerelease intended only for testing. It may contain
> bugs. Back up your Word Hunter data before installing and report regressions
> through GitHub Issues.

## Changes Since RC2

- Familiar and Alternative Familiar now remain dark on desktop from the first
  paint, including the offline dictionary and YouGlish integrations. Pocket
  retains system-responsive light and dark variants.
- Swiping left or right in the Reader while the selected-word panel is open now
  selects the next or previous visible token and animates the panel direction.
- Flashcards now use a distinct layered-card surface. Tapping the card reveals
  the answer; swiping left or right moves through the ungraded review queue.
  Navigation does not grade cards or modify their SRS schedule.
- Heatmaps in Pocket Graphs and Flashcards initially align to the latest weeks
  while preserving manual horizontal scrolling across Flashcard rerenders.
- Moved Flashcard analytics farther below the primary deck and retained explicit
  grading controls after answer reveal.

## Included From Earlier RCs

- Complete TypeScript browser runtime and deterministic generated frontend.
- Configurable selected-word panel visibility and ordering.
- Desktop Edit Book layout corrections and cross-platform reliability fixes.

## Update Policy

GitHub marks this build as a prerelease. Stable Word Hunter installations query
only GitHub's latest stable release endpoint, so this test build does not trigger
the in-app new-version popup.
