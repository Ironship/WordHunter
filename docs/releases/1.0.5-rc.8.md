# Word Hunter 1.0.5-rc.8

> **Warning:** This is a prerelease intended only for testing. It may contain
> bugs. Back up your Word Hunter data before installing and report regressions
> through GitHub Issues.

## Changes Since RC7

- The Pocket selected-word sheet now derives its height from the active drag
  position. Expanded, collapsed, and intermediate positions keep the complete
  usable area above the bottom navigation instead of clipping lower controls.
- The grammatical Article editor is now a regular selected-word panel item. It
  is visible by default and can be hidden or reordered with Status, Dictionary,
  Speech, Notes, and the other configurable items.
- The selected-word item ordering controls in Reader settings can be collapsed,
  keeping the settings page compact when the full list is not being edited.
- Article controls use a bounded desktop width and a responsive Pocket layout,
  improving their fit alongside Smart Suggestion actions.
- Background synchronization no longer rebuilds an unchanged active Reader.
  This prevents the reading screen from flashing and jumping to the top while
  the user is idle, resolving GitHub issue #63. A synchronized text is still
  refreshed when its body actually changes.
- Frontend release validation again counts the complete generated TypeScript
  runtime and exercises Article rendering from the correct word-panel target.

## Included From Earlier RCs

- Grammatical-article persistence, suggestions, display, speech, search,
  flashcards, and import/export support.
- Synchronization revision guards and exact Ctrl+Enter Reader restoration.
- Extended Pocket word-sheet gestures, easier card swipes, and reachable
  navigation on tall phones.
- Complete TypeScript browser runtime and deterministic generated frontend.
- Desktop theme, Edit Book, heatmap, and SRS consistency fixes.

## Update Policy

GitHub marks this build as a prerelease. Stable Word Hunter installations query
only GitHub's latest stable release endpoint, so this test build does not trigger
the in-app new-version popup.
