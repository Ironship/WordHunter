# Word Hunter 1.0.5-rc.1

> **Warning:** This is a prerelease intended only for testing. It may contain
> bugs. Back up your Word Hunter data before installing and report regressions
> through GitHub Issues.

## Highlights

- Migrated all shipped browser runtime code to TypeScript, including the 87
  existing modules and the boot and translator-popup scripts previously inline
  in HTML.
- Added a deterministic `dist/web` build consumed by Windows, Android, and
  Flatpak packaging.
- Added visibility checkboxes and accessible Up/Down ordering controls for the
  selected-word panel.
- Moved dictionary, speech, YouGlish, and remove actions directly below word
  status controls by default.
- Added optional Copy and Edit actions and improved Pocket panel positioning.
- Fixed theme persistence, state normalization, contrast, chart colors, and
  offline-translator theme handling.
- Fixed dialog, drawer, clipboard, image-action, and asynchronous panel edge
  cases found during migration review.
- Completed missing interface translations and added stronger localization
  coverage.

## Update Policy

GitHub marks this build as a prerelease. Stable Word Hunter installations query
only GitHub's latest stable release endpoint, so this test build does not trigger
the in-app new-version popup.

## Downloads

- Windows installer: `Word.Hunter.Setup.exe`
- Windows portable archive: `Word.Hunter.portable.zip`
- Android test APK: `Word.Hunter.Pocket.debug.apk`
- Android unsigned validation bundle: `Word.Hunter.Pocket.release.aab` (not
  directly installable and not intended for a stable store track)
- Linux bundle: `WordHunter.flatpak`
