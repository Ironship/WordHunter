# Word Hunter 1.0.5-rc.2

> **Warning:** This is a prerelease intended only for testing. It may contain
> bugs. Back up your Word Hunter data before installing and report regressions
> through GitHub Issues.

## Changes Since RC1

- Fixed light desktop themes using nearly identical white backgrounds. Classic,
  Familiar, and Alternative Familiar now use distinct theme-matched desktop
  backgrounds and panel surfaces while Pocket styling remains unchanged.
- Fixed the desktop Edit Book dialog allocating most of its width to labels and
  squeezing inputs, cover controls, and the text editor into a narrow column.
- Added regression coverage for desktop theme surfaces, contrast, responsive
  Edit Book layout, and unchanged Pocket behavior.

## Included From RC1

- Migrated all shipped browser runtime code to TypeScript.
- Added deterministic frontend builds for Windows, Android, and Flatpak.
- Added configurable visibility and ordering for the selected-word panel.
- Improved Pocket panel positioning, localization, dialogs, drawers, themes,
  updater version handling, and asynchronous interaction safety.

## Update Policy

GitHub marks this build as a prerelease. Stable Word Hunter installations query
only GitHub's latest stable release endpoint, so this test build does not trigger
the in-app new-version popup.

## Downloads

- Windows installer: `Word.Hunter.Setup.exe`
- Windows portable archive: `Word.Hunter.portable.zip`
- Android test APK: `Word.Hunter.Pocket.debug.apk`
- Android unsigned validation bundle: `Word.Hunter.Pocket.release.aab`
- Linux bundle: `WordHunter.flatpak`
